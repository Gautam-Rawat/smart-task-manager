# Smart Task Manager

A full-stack task management application that helps users organize and track daily tasks with authentication and personalized dashboards.

## 🚀 Features
- User authentication (JWT-based login/signup)
- Create, update, delete tasks (CRUD)
- Personalized dashboard for each user
- Responsive UI for all devices

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT

## 📂 Project Structure
client/ → React frontend  
server/ → Backend APIs  

## ⚙️ Setup Instructions

### Clone the repository
git clone https://github.com/Gautam-Rawat/smart-task-manager.git

### Install dependencies
cd client && npm install  
cd ../server && npm install  

### Run the app
cd server && npm start  
cd ../client && npm start
